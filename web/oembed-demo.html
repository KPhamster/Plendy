The Facebook Reel above is rendered using Meta's oEmbed Read API:

  • Plendy calls graph.facebook.com/v21.0/oembed_video
  • The API returns HTML embed code for the video
  • The embed is rendered using the Facebook SDK
  • Users can then extract location data and save to their experiences
